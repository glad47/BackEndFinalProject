登录接口

请求URL

/api/auth/login POST

请求参数 application/x-www-form-urlencoded

参数名 类型 必须 描述
username string 是 用户邮箱
password string 是 用户密码
recaptchaResponse string 是 google校验码 reCaptcha,返回的g-recaptcha-response

返回结果

{
	"success":"boolean //是否成功",
	"code":"string //编码",
	"message":"string //描述信息",
	"result":{}
}
